# Ret20221 Compiler (UFRGS INF01147)

This repository contains a compiler implementation for the `ret20221` language created as part of the UFRGS Compilers course (INF01147) taught by Professor Marcelo Johann.

## Authors:

- [Arthur Prochnow Baumgardt](https://github.com/4rthb)
- [Tiago Lucas Flach](https://github.com/tiagoFlach)

## Project Overview

The compiler includes the following phases:

- Lexical analysis using `flex` (`scanner.l`)
- Syntax analysis using `bison`/`yacc` (`parser.y`)
- Symbol table management with a hash table (`hash.c` / `hash.h`)
- Abstract syntax tree (AST) creation and traversal (`ast.c` / `ast.h`)
- Semantic analysis and error checking (`semantic.c` / `semantic.h`)
- Three-address code (TAC) generation and assembly output (`tacs.c` / `tacs.h`)
- Driver program with file I/O and pipeline orchestration (`main.c`)

## Repository Layout

- `main.c` — compiler driver, command line handling, and compilation pipeline
- `scanner.l` — lexical rules and token generation
- `parser.y` — grammar, AST construction, and syntax error handling
- `hash.c` / `hash.h` — symbol table implementation and temporary label generation
- `ast.c` / `ast.h` — AST node creation, printing, and decompilation
- `semantic.c` / `semantic.h` — semantic validation and type checking
- `tacs.c` / `tacs.h` — TAC generation, TAC printing, and assembly output
- `Makefile` — build targets for compilation and cleanup
- `test/` — example source programs and expected output files

## Supported Language Features

The implemented language supports:

- Primitive types: `char`, `int`, `float`
- Global variable and vector declarations
- Function declarations and calls
- Arithmetic operators: `+`, `-`, `.`, `/`
- Comparison operators: `<`, `>`, `<=`, `>=`, `==`, `!=`
- Logical operators: `&`, `|`, `~`
- Control flow: `if`, `if ... else`, `while`
- I/O commands: `read`, `print`
- `return` statements in functions
- Assignments using `<-`
- String literals and character literals
- Single-line comments beginning with `##`
- Multi-line comments between `\#` and `#\`

## Build Instructions

This project requires:

- `flex`
- `bison`
- `gcc`

Build the compiler with:

```bash
make
```

Run cleanup with:

```bash
make clean
```

## Usage

Compile and execute a source file with:

```bash
./compiler input.txt output.txt
```

- `input.txt` is the source program in the compiler language
- `output.txt` receives the decompiled AST output / printed program representation
- `output.s` is generated by the code generation phase as x86-64 assembly

The compiler also prints symbol table contents and AST structure to `stderr`.
